# Book Recommendation System 📚

A machine learning-powered book recommendation system built using React, FastAPI, NumPy, and Pandas.

Recently, I wanted to understand how platforms recommend content based on user behavior and preferences. To explore recommendation systems, I built this project using a large-scale book dataset and implemented both popularity-based and collaborative filtering approaches to generate personalized recommendations.

## Live Demo

🚀 https://book-recommendation-system-3-engn.onrender.com/

## Dataset

The recommendation engine was built using a dataset containing:

* 271K+ Books
* 1.1M+ Ratings
* 278K+ Users

## Features

⭐ Popularity-based book recommendations

⭐ Collaborative filtering recommendations

⭐ User-book interaction matrix generation

⭐ Similarity-based recommendation engine

⭐ FastAPI backend for recommendation serving

⭐ React frontend for user interaction

⭐ Pre-trained recommendation models using Pickle

⭐ Real-time recommendation generation

## Key Engineering Highlights

* Processed and analyzed a dataset containing over 1 million ratings
* Built popularity-based recommendation logic using rating count and average rating thresholds
* Developed collaborative filtering recommendations using user-book interaction matrices
* Implemented cosine similarity for identifying similar books
* Serialized recommendation models using Pickle for efficient deployment
* Integrated React frontend with FastAPI backend to serve recommendations in real time

## Recommendation Approaches

### Popularity-Based Recommendation

Books are recommended based on:

* Number of ratings
* Average rating score
* Minimum popularity thresholds

### Collaborative Filtering

Recommendations are generated by:

* Creating user-book interaction matrices
* Computing cosine similarity between books
* Finding books with similar user engagement patterns
